# 给同事使用（无需安装 Python）

## 你这边先做一次打包

1. 双击运行 `build_exe.bat`
2. 成功后会生成：`dist/PDF2Excel-WebUI.exe`

## 发给同事的内容

只需要发这一个文件：`PDF2Excel-WebUI.exe`

## 同事如何使用

1. 双击 `PDF2Excel-WebUI.exe`
2. 会自动打开网页界面（通常是 `http://127.0.0.1:5000`）
3. 拖入 PDF，点击“开始转换并下载”
4. 浏览器会直接下载 Excel

## 说明

- 转换逻辑只处理前 4 页（自动跳过第 5 页图形页）
- 文件在本机临时目录处理，不会长期保存在程序目录
- 下载到哪里由浏览器下载设置决定

## 常见问题

- 如果双击后没有打开页面：手动访问 `http://127.0.0.1:5000`
- 如果端口被占用，可命令行运行：
  `PDF2Excel-WebUI.exe --port 5001`
